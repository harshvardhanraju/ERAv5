Experiment 01

Activations Exist for a Reason

A model with no nonlinearity can only draw a straight line. It cannot separate two concentric rings. Adding one ReLU hidden layer lets the boundary wrap around the inner ring — accuracy jumps from ~55% to ~99%.

Key insight: The linear model's boundary is literally a straight line — the weights can only define a hyperplane in 2D. The ReLU model pieces together multiple linear regions, carving out the inner ring. Only the activation changed. Everything else is identical.
Experiment 02

Depth Without Nonlinearity Is a Lie

Five stacked linear layers mathematically collapse into one linear map. A 5-layer linear network is no stronger than a 1-layer network — both draw the same straight-line boundary. Insert ReLUs and the five layers suddenly solve it.

5-Layer Linear —

1-Layer Linear 2→1 · sigmoid
5-Layer Linear no activations
5-Layer ReLU ReLU between layers
W₁ × W₂ × W₃ × W₄ × W₅ = One Single Matrix

The product of all five weight matrices is just one 2×1 matrix — equivalent rank to a single linear layer.

💡
Key insight: W₁W₂W₃W₄W₅ = W_eff — a single linear transformation. No matter how deep you stack linear layers, composition of linear functions is still linear. Nonlinearity is what makes depth meaningful.

Experiment 04

Memorization vs Generalization

An over-parameterized model on tiny data drives training loss to zero while test loss stays high — it memorizes, not learns. Growing the dataset closes the gap. Data is everything.

Key insight: With n=20, the model has enough parameters to memorize every training point, so train loss → 0 but test loss stays high. With n=2000, the sheer variety of examples forces the model to learn the underlying structure. You cannot regularize your way out of too little data.
